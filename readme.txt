Minimum Dimensions For Image Field WordPress Plugin

== Description ==
This plugin allows the user to set size constrain for the "Featured Image" field.

= Plugin Features =

*   Add size constrain for the "Featured Image" field.
*   Just go to Appearance -> "Media Settings" and add your desired constrains.
*   Ability to add size constrain for custom fields( ACF ).
*   Currently not working with pods. very soon the feature will be added for pods

== Installation ==

1. Extract the `st_media.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Note: If wp_debug option is enabled, then this plugin wont work.It conflicting with wp_debug.So make sure wp_debug is disabled for production.
